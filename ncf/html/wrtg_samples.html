User Guides

A user guide contains information about how to install, configure, and make use of a hardware device, system, product, or software application, although sometimes installation, configuration, and troubleshooting instructions might be presented in separate documents. The intended audience for such information is anyone who needs to carry out these tasks, so the type of information in a guide can be quite varied. User guides might contain conceptual and reference information as well as sets of steps, or procedures, for performing common operations.

The user guide samples here are examples of two methods of presenting technical information: as single PDF documents (or extracts of PDF documents), and as sets of individual HTML files (topics) that comprise the guide. The latter method enables the documentation to be integrated into the help system of an application or software tool and to be read online. Presenting information in a PDF document enables users to download the file for offline viewing or printing.
